Waldis Pathfinder Campaign

The Waldis Pathfinder campaign is an ongoing project which involves creating an entire series of challenges for my group of players. Before you simply skip this as not relevant, consider how much management and organization it takes to manage a campaign spanning not months, but years. I've been designing everything from the very start, building battles, creating social interactions, balancing loot and difficulty. I've had to adjust entire sections as the players choose options I didn't consider, making entirely new story trees as a result. Not only is there a lot of different things to organize behind the scenes but then I actively perform them infront of my player group. During the performance I need to follow what I set and then adjust to react to what the players have decided to do. All of this needs to be done following the rules of the game, while making it fun for everyone involved.

Flash Programs

I've created quite a few Flash programs as the Flash application easily lets you combine code and artwork in a single program.

Fireworks Display

This Flash program displays randomly colored fire work effects chosen from a few different effect types. This program uses packages to manage the seperate types of fireworks.




Face Maker

The Face Maker program allows the user to create character portraits with a good amount of customization. The portraits can be exported as png image files..




Pathfinder Monster Form

This Flash program works as a form to create a monster for the Pathfinder tabletop game. Some of the stats are generated automatically and once complete the user can copy the information to the clipboard to paste them into a word processor for futher editing or printing.
