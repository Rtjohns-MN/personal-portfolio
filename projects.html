TAPS Documentation

The Teaching Assistant Processing System (TAPS) documentation was created in the CS5801 Software Engineering 1 course. During the course I was assigned to a group and we were tasked to create the documentation to encompass the TAPS user requirements and the design specifications.

Tabletop Manager

The Tabletop Manager program was created in the CS5115 User Interface Design. We were assigned a group and allowed to choose the application to build for the class. The group settled on creating an application which assisted with managing the character sheet for the Dungeons and Dragons 5th edition game. The app was built in Android Studio over roughly 3 weeks.

Commando Shell Program

Commando is a shell program created in the Linux operating system using the C language. It allows users to run other programs, tracks their status and can read their outputs when they are completed.
